Questo comando "SFTP", è utilissimo.

Se ssh serve per collegarsi al terminale di qualche computer (lan/internet), SFTP server per fare una cosa simile ma con la possibilità di scaricare e/o caricare i file.

è utilissimo e si usa alla stessa maniera di ssh:

- ssh user@address
- sftp user@address

usano anche i comandi simili, come CD per muoversi tra le cartelle.

un comando utilissimo è PUT, che in sostanza prende, dalla cartella in cui sei, un file desiderato e poi lo mette nella directory desiderata nel server/computer desiderato.

molto utile per caricare e scaricare i file senza difficoltà.

Per tutti i comandi devo finire di leggere, studiare e provare tutto quando dal link postato all'inizio.
